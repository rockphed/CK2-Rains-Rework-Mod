Rework of African Rains feature of Crusader Kings 2

Features include:
  * New events for the African Rain features
  * Reworking events so they are less punishing
  * Making the rain events tie into the world better (future)
  * Possibly adding weather events to other parts of the map (distant future)

Includes event file that overrides the African Rains events, a localization file that localizes things in English for same, and a couple of modifiers.

So far, coded entirely by Rockphed
